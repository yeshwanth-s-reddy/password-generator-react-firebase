Secure Password Generator
A modern, cloud-based password generator and manager built with React.js, Clerk authentication, and Firebase Realtime Database.

 Features
- Generate secure customizable passwords
- Real-time password strength meter
- User authentication (Clerk)
- Cloud-based password history (Firebase)
- Cross-device access
- Secure user-specific data storage

Tech Stack
- React.js (Vite)
- Tailwind CSS
- Clerk Authentication
- Firebase Realtime Database
- JavaScript (ES6+)

Security Features
- User-specific password storage
- Firebase security rules
- Session-based authentication
- No backend server exposure

Installation1. Clone the repo:
   git clone https://github.com/yourusername/secure-password-generator.git

2. Install dependencies:
   npm install

3. Add your Clerk & Firebase keys in `.env`

4. Run:
   npm run dev
